# Oracle JDK Ansible playbook

[![NepLab](http://neplab.com/wp-content/uploads/2016/02/neplab-logo-retina.png)](https://support.neplab.com) 


## Description
Install JDK 1.8 or above on Linux Box


## Compatibility

only works with RHEL 64 bit > 6.9 OS Version

## Usage

- Clone the main.yml file
- Change the java version to the one that you'd like
- Execute command:

ansible-playbook -inventory DEV main.yml
